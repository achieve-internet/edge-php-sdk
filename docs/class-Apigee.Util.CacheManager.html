Class CacheManager

Cache classes must extend this class and provide a default constructor, Factory class will create an instance of such class and then invoke setup method, setup method is passed as arguments what is returned by getConfig method, setup logic can then be performed out of the constructor. Notice that setup should only be invoked once and only by the Factory class
