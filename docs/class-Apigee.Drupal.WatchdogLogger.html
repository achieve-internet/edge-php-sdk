Class WatchdogLogger

A logger class that implements Psr\Log\LoggerInterface. This class writes log data to Drupal's watchdog table.

Psr\Log\AbstractLogger

# setLogThreshold( integer $threshold )
Sets the logging threshold, below which log entries will be dropped rather than logged, as definedby RFC 3164: http://www.faqs.org/rfcs/rfc3164.html. These values are implemented by Psr\Log\LogLevel as:
  • LogLevel::EMERGENCY
  • LogLevel::ALERT
  • LogLevel::CRITICAL
  • LogLevel::ERROR
  • LogLevel::WARNING
  • LogLevel::NOTICE
  • LogLevel::INFO
  • LogLevel::DEBUG
